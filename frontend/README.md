# Frontend — Sanctions Entity Explorer

React 19 + TypeScript + Vite. Strict TS, `noUncheckedIndexedAccess` on. Build on top — replace anything that doesn't fit.

## Setup

```bash
cd frontend
pnpm install   # or: npm install / yarn install
```

## Run

```bash
pnpm dev
```

App at http://localhost:5173. Backend expected at http://localhost:8000.

## Build

```bash
pnpm build
```

## Type-check

```bash
pnpm typecheck
```
